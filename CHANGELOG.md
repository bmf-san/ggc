## [Unreleased]

### Added

- `ggc version json` emits machine-readable version/commit/build metadata
  for CI and scripting.
- `ggc doctor` now also warns about shadowed `ggc` binaries on PATH,
  older-than-2.30 Git installs, `TERM=dumb`/unset, and missing fish
  completions.
- Release artifacts are now signed with cosign keyless (Sigstore); see
  the [installation guide](guide/install.md#verifying-a-downloaded-release).
- New [Recipes](guide/recipes.md) page collects task-oriented workflows
  (start a feature branch, fixup + autosquash, stash a WIP, tag a
  release, …).

### Changed

- Command reference on the docs site now uses stable anchors (`#cmd-<name>`
  and `#cat-<category>`) so deep links from external pages remain valid
  across regenerations.
- The MkDocs search plugin now indexes Japanese as well as English.
